# Project Brief: Vibe Code Academy

## Product Idea

Vibe Code Academy is a practical vibe coding workshop that helps people turn rough product ideas into simple AI-assisted prototypes.

## Target User

Students, marketers, and early-stage founders who have product ideas but limited coding experience.

## Problem

The target user has ideas, but gets stuck between talking about the idea and making something real. AI tools are powerful, but they do not know what to ask for, how to structure the work, or how to judge whether the output is useful.

## Promise

In one focused workshop, Vibe Code Academy helps you turn a rough idea into a simple prototype plan, first build, and next-step roadmap using Cursor and AI-assisted workflows.

## Brand Direction

Practical, friendly, no-hype, beginner-friendly, and clear. The page should feel modern but not overly flashy.

## Primary Goal

Get visitors to enter their email to receive the Prototype Starter Kit.

## Prototype Starter Kit

The starter kit includes:

- a prototype planning template
- a build prompt template
- a simple project file structure

## Core Page Sections

- Hero section with clear promise
- Problem section
- What you will build
- How the workshop works
- What is included in the starter kit
- Email signup form
- FAQ or reassurance section

## Functional Requirement

The page should include an email signup form. For this prototype, the form can show a success state after submission. A more advanced version would save the email to a JSON file, SQLite database, or external email platform.

## In Scope

Single-page landing page, email signup form, simulated success state, polished but simple visual design.

## Out of Scope

Payment, user accounts, real email delivery, admin dashboard, authentication, video hosting, full CMS, and production deployment.

## Success Criteria

A visitor should understand who Vibe Code Academy is for, what they will get, why it is useful, and what action to take within 60 seconds.
