# Value Proposition: Vibe Code Academy

## One-Line Value Proposition

Vibe Code Academy helps you turn a rough product idea into a visible, testable prototype using AI tools, especially Cursor, without needing to become a software engineer first.

---

## Short Value Proposition

Most people have ideas, but get stuck when it is time to make those ideas real.

Vibe Code Academy teaches a practical AI-assisted workflow for moving from idea to prototype:

**Plan → Build → Test → Improve → Share**

You learn how to define the user, clarify the problem, create a project brief, guide Cursor, review the output, and understand what is real versus simulated in your prototype.

The outcome is not a finished software product.

The outcome is a prototype artifact you can show, test, explain, and improve.

---

## What the Class Provides

### 1. A Clear Mental Model For AI-Assisted Building

The class teaches you what vibe coding is, why it works, and where it breaks.

You learn that AI is not a replacement for clear thinking. AI is an execution layer and coach. It can help you move faster, but only if you provide direction, context, constraints, and feedback.

---

### 2. A Practical Planning Framework

Before building, you learn how to define:

- Target user
- Problem
- Prototype goal
- Desired outcome
- In-scope features
- Out-of-scope features
- Success criteria

This helps prevent the most common AI-building failure: asking the tool to build before you know what you are trying to prove.

---

### 3. A Guided Cursor Workflow

You learn how to move out of a chat box and into a real project folder.

The class introduces how to use Cursor to:

- Organize project files
- Create a project brief
- Create a build plan
- Generate a first prototype
- Review and improve AI-generated work
- Preserve context across build sessions

---

### 4. A Simple Prototype Artifact

You leave with a simple prototype connected to your own idea.

Depending on your project, this may be:

- A startup landing page
- A dashboard or report mockup
- A portfolio site
- A simple AI-assisted product concept
- Another scoped prototype that fits your idea

The goal is to make the idea visible enough to test, discuss, and improve.

---

### 5. Feedback and Iteration

You test your prototype with another person and use feedback to improve it.

This helps you learn that a prototype is not just something you build. It is something you use to learn.

---

### 6. A Repeatable Workflow

You leave with a process you can reuse after the class:

1. Clarify the idea.
2. Write the project brief.
3. Break the build into phases.
4. Build one phase at a time.
5. Test the output.
6. Document what changed.
7. Decide the next step.

---

---

## What This Is Not

Vibe Code Academy is not a promise that AI will build a complete software company for you.

It is not a replacement for engineering, testing, security, databases, deployment, or product strategy.

It is a practical starting point for making ideas visible and learning how to guide AI more effectively.

---

## Why It Matters

AI makes it easier than ever to create something that looks polished.

That is useful, but also dangerous.

A polished screen can hide weak thinking, fake functionality, unclear users, and missing product logic.

Vibe Code Academy helps you avoid that trap by teaching you how to plan before building, test what you create, and understand what still needs to be done.
