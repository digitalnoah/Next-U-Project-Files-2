Vibe Code Academy

A practical academy for AI-assisted builders. Less “magic AI startup.” More craft, code, critique, and repeat.

01 / Position

Serious enough to trust. Weird enough to remember.

The brand should feel like a modern digital studio teaching practical AI prototyping. It should be confident, stripped down, and sharp, while still being welcoming to beginners.

Promise

From idea to artifact.

Help learners turn rough ideas into visible prototypes they can test, explain, and improve.

Audience

Smart beginners.

Students, marketers, founders, and operators who have ideas but do not yet have a repeatable AI build workflow.

Attitude

No guru energy.

Teach the craft. Show the work. Admit the limits. Make the next step obvious.

02 / Logo

A small website mark for a builder academy.

The logo system uses a compact code mark and a horizontal wordmark. The small icon is best for favicons, cards, and tight website spaces. The full lockup works best in a header or brand presentation.

Primary Horizontal Lockup
Vibe Code Academy

Use the mark with the wordmark when space allows. Use the icon alone when the brand already has context.

03 / Palette

Black, bone, acid. Then one digital accent.

Keep the palette restrained. Most pages should be black, bone white, and warm neutrals. Use acid green as the signature. Blue and coral are secondary accents only.

04 / Voice

Confident, plainspoken, slightly mischievous.

The voice should sound like a smart instructor who has actually built things. It should be direct, useful, and a little memorable without becoming cute.

Rule Use This Avoid This
Lead with the artifact Leave with a prototype you can show, test, and improve. Unlock your AI transformation journey.
Admit the constraint AI can build fast. It still needs direction. Let AI build the whole thing for you.
Make the craft visible Plan the work. Build one phase. Test it. Repeat. Prompt better and watch the magic happen.
Stay human You do not need to be an engineer to start. You do need to think clearly. No-code agentic full-stack workflow mastery.
05 / Messaging

Sample copy with the right edge.

Build the thing. Then make it better.

Vibe Code Academy teaches you how to turn a rough product idea into a prototype with AI. You will learn how to plan the work, guide Cursor, test the output, and avoid confusing a polished screen with a real product.

The goal is not to become a software engineer overnight. The goal is to become a better director of AI-assisted work.

06 / Builder Notes

How Cursor should use this brand guide.

Use this file as visual and voice direction when building the Vibe Code Academy landing page. Do not overdecorate. Build a strong grid, high contrast, direct copy, and a clear signup path.

Design direction

Dark editorial layout, strong typography, modular sections, large whitespace, sharp CTAs, restrained animation, no glossy SaaS gradients.

Page structure

Hero, problem, workshop outcome, how it works, starter kit, example projects, signup, FAQ.

When building the landing page, follow this brand guide: - Use Studio Black, Bone White, and Signal Acid as the main palette. - Use large, confident typography. - Keep the tone practical, direct, and no-hype. - Make the signup CTA obvious. - The form can simulate success, but must clearly distinguish simulated behavior from production functionality. - Avoid generic SaaS copy and generic AI hype.
