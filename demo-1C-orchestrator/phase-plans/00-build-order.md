# Build Order — Vibe Code Academy Landing Page

Execute **one phase per session**. Read the build file, implement, then run QA and human validation before advancing.

| Phase | Build file | Focus |
|-------|------------|--------|
| 1 | `build/phase-01-build.md` | Content architecture and semantic HTML skeleton |
| 2 | `build/phase-02-build.md` | Brand design system (CSS tokens, typography, layout) |
| 3 | `build/phase-03-build.md` | All page sections, copy, pricing, FAQ, CTAs |
| 4 | `build/phase-04-build.md` | Email signup form, validation, success state |
| 5 | `build/phase-05-build.md` | Responsive layout, accessibility, polish, single-file consolidation |

**Start here:** `build/phase-01-build.md`

**Do not** skip phases. **Do not** build future phases while executing an earlier one.
